# Java-Based Hospital Management System

Java application developed for managing hospital activities, including medical staff, patients, appointments, consultations, and hospitalizations.

## Features
- Manage patients and medical staff
- Schedule appointments and consultations
- Handle hospitalizations
- Support CNAS reimbursement and insurance-based pricing
- Doctors' scheduling management
- Graphical user interface

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and object relationships
- GUI development
- Data management and application logic

## Project Structure
- `/java-based-hospital-management-system` – Java source code
- `project-documentation.pdf` – project documentation

## How to Run
1. Open the project in IntelliJ IDEA
2. Build the project
3. Run the main class

## Notes
This project was developed as part of a university assignment and focuses on applying OOP principles in a practical hospital management application.
